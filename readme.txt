⚡ Electricity Demand Forecasting
📌 Overview
This project predicts hourly electricity demand using machine learning models like Linear Regression, Random Forest, and XGBoost.

📊 Data Source
U.S. Energy Information Administration (EIA API)

⚙️ Features Used
Hour, Day, Month, Weekday

Lag Features (lag_1, lag_24)

Rolling Mean (24-hour)

🤖 Models
Linear Regression

Random Forest

XGBoost

📈 Results
Achieved high accuracy (R² ~ 0.92+)

XGBoost performed best

🚀 How to Run
pip install -r requirements.txt
python main.py
